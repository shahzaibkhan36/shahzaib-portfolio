## 🚀 How to Deploy on Vercel (Recommended — FREE)

**Vercel is the best platform for this.** It's fast, free, gives a live URL instantly, and auto-redeploys every time you push to GitHub.

### Step 1 — Put files in a folder on your PC
Download `index.html` and `README.md` and place them inside a folder named `shahzaib-portfolio`.

### Step 2 — Create GitHub repository
1. Go to → [github.com/new](https://github.com/new)
2. **Repository name:** `shahzaib-portfolio`
3. **Visibility:** Public
4. Do **NOT** tick "Add a README file" (you already have one)
5. Click **Create repository**

### Step 3 — Push to GitHub
Open **Git Bash** or your terminal, navigate into your folder, then run:

```bash
git init
git add .
git commit -m "🚀 Initial commit — portfolio website"
git remote add origin https://github.com/shahzaibkhan36/shahzaib-portfolio.git
git branch -M main
git push -u origin main
```

### Step 4 — Deploy on Vercel
1. Go to → [vercel.com](https://vercel.com)
2. Click **Sign Up** → choose **Continue with GitHub**
3. Click **Add New Project**
4. Find and click **Import** next to `shahzaib-portfolio`
5. Framework Preset → select **Other**
6. Click **Deploy**
7. ✅ Your site is LIVE! You'll get a URL like `shahzaib-portfolio.vercel.app`

> 💡 Update the Live Site link at the top of this README with your actual Vercel URL.
